Aprendendo HTML & CSS

Você pode escrever o que quiser, porém precisa prestar atenção na quebra nas tags.
Não é dificil, apenas pratica, precisa olhar o código como se estivesse olhando o front-end do site.
Dessa forma você consegue aprender de forma mais precisa.

Observação: se precisar quebrar o texto em algum lugar, você precisa usar a tag br.

Vamos adicionar simbolos no seu código,
exemplos: ® © ™ € £ ¢ δ

Hora de treinar colocando vários emojis no código,
exemplos: 😀 😃 😄 😁 😆 🖖

Falando sobre direitos autorais 😁

Não é correto você pegar qualquer imagem na internet para inserir no seu site.
Para você pegar imagens de forma legal é necessário realizar uma busca no google com o nome na imagem que deseja, ir na opção "ferramentas" em seguida em "direitos de usos" e selecionar a opção "marcas para reutilização com modificação.", existe outras opções mas é necessário verificar. Essa opção permite pegar imagens de forma legal do google
